# HTML-Generator beta

Dieser Generator bündelt den Quellcode der aktuellen Version des OpenElectionCompass, aller zugehöriger Ressourcen und eine bereitgestellte Konfigurationsdatei in einer einzelnen HTML-Datei. Diese Datei kann anschließend leicht auf einem einfachen Webserver bereitgestellt werden. Diese Deployment-Methode ist nicht ohne Nachteile, es ist jedoch sicherlich die einfachste selbstgehostete Lösung. Erzeuge einfach eine Konfigurationsdatei mithilfe des Konfigurations-Editors und lade sie in diesen Generator.

1. Upload configuration file
A configuration file in the JSON file format. You can write one from scratch, or you can use the Configuration Editor.
